# Ανάπτυξη Εφαρμογής Chatbot για την Ανάλυση Δεδομένων από τη Διαύγεια 

 
## Ακαδημαϊκό Πλαίσιο 
Η παρούσα εργασία εκπονήθηκε στο πλαίσιο **Διπλωματικής Εργασίας** για το **Πανεπιστήμιο Δυτικής Μακεδονίας**, Τμήμα Οργάνωσης και Διοίκησης Επιχειρήσεων, στο Μεταπτυχιακό Πρόγραμμα Σπουδών στη Διοίκηση Επιχειρήσεων (MBA) με κατεύθυνση τα Πληροφοριακά Συστήματα Διοίκησης (MIS). 

## Περιγραφή Έργου 
Η εφαρμογή αποτελεί μια συνομιλιακή διεπαφή (Chatbot) βασισμένη στο framework **Streamlit**. Επιτρέπει την ανάκτηση και στατιστική ανάλυση διοικητικών πράξεων από το Open Data API της πλατφόρμας "Διαύγεια", προσφέροντας μια φιλική προς τον χρήστη εμπειρία αναζήτησης μέσω φυσικής γλώσσας. 

## Κύριες Λειτουργίες 
- **Αναζήτηση Οργανισμών:** Εύρεση και επιλογή δημόσιων φορέων μέσω αναζήτησης **Επωνυμίας** ή απευθείας μέσω **ΑΦΜ** για την ανάκτηση του μοναδικού κωδικού (UID). 
- **Εξειδικευμένη Ανάλυση (Analytics):** 
  - **Δ.1 (Ανάθεση έργων/ Προμηθειών/ Υπηρεσιών/ Μελετών):** 
    -  Υπολογισμός συνολικού χρηματικού ποσού και πλήθους συμβάσεων ανά ανάδοχο. 
    - Άθροισμα ποσών ανά τύπο ανάθεσης (π.χ. Προμήθειες, Υπηρεσίες).
  - **Β.2.1 (Έγκριση Δαπάνης):** Υπολογισμός του αθροίσματος των χρηματικών ποσών πληρωμής ανά άτομο ή επιχείρηση και καταγραφή του συνολικού πλήθους των σχετικών πράξεων έγκρισης.
  - **Β.2.2 (Οριστικοποίηση Πληρωμής):** Άθροισμα του χρηματικού ποσού πληρωμής ανά δικαιούχο (φυσικό πρόσωπο ή εταιρεία) και υπολογισμός του πλήθους των αντίστοιχων πράξεων οριστικοποίησης.
  - **Β.4 (Δωρεά - Επιχορήγηση):** Καταγραφή και ομαδοποίηση του πλήθους των δωρεών ανά αποδέκτη. 
  - **Β.5 (Παραχώρηση Χρήσης Περιουσιακών Στοιχείων):** Στατιστική απεικόνιση του πλήθους των παραχωρήσεων ανά δωρεοδόχο.
  - **Γ.3.5 (Υπηρεσιακή Μεταβολή):** Ανάλυση του πλήθους των πράξεων ανά είδος υπηρεσιακής μεταβολής (π.χ. Αποδοχή Παραίτησης, Λύση Υπαλληλικής Σχέσης, Μετάταξη).
- **Διαδραστικό Chat:** Μια φιλική ροή εργασίας (Step-by-step) για το φιλτράρισμα των αποτελεσμάτων. 
- **Οπτικοποίηση Δεδομένων:** Χρήση δυναμικών γραφημάτων (**Altair**) για την παρουσίαση των οικονομικών στοιχείων. 
- **Αναζήτηση με ΑΔΑ:** Άμεση ανάκτηση και μορφοποίηση όλων των πεδίων μιας συγκεκριμένης πράξης. 

## Δομή του Project 
Ο κώδικας έχει οργανωθεί σε αρχεία  για βέλτιστη αναγνωσιμότητα και συντήρηση: 
1.  **`app.py`**: Ο "πυρήνας" της εφαρμογής. Περιλαμβάνει τη διαχείριση της διεπαφής (UI) και τη λογική της συνομιλίας (Chat Logic). 
2.  **`api_handler.py`**: Διαχειρίζεται όλη την επικοινωνία με το API της ΔΙΑΥΓΕΙΑΣ (requests, caching, pagination). 
3.  **`analysis.py`**: Περιλαμβάνει τις εξειδικευμένες συναρτήσεις επεξεργασίας δεδομένων και παραγωγής γραφημάτων ανά τύπο απόφασης. 
4.  **`helpers.py`**: Περιλαμβάνει βοηθητικές συναρτήσεις. 

##  Οδηγίες Εγκατάστασης & Χρήσης 
1. Βεβαιωθείτε ότι έχετε εγκατεστημένη την Python 3.8+. 
2. Κάντε clone το repository και εγκαταστήστε τις απαραίτητες βιβλιοθήκες 
3. Εκτέλεση 
```bash 
streamlit run app.py 
```
